# Contributing

Thanks for contributing to the Kaspa Sovereign ecosystem work.

## How To Contribute

1. Fork the repository.
2. Create a branch for your improvement.
3. Keep changes focused and production-ready.
4. Open a pull request with:
   - what changed
   - why it matters
   - how it was validated

## Join And Expand

This project is open to builders who want to expand the skill engine, adapters, tooling, docs, testing, and infrastructure.

- Improve compatibility across AI platforms.
- Add reliability, security, and scaling enhancements.
- Expand Kaspa-specific architecture and developer workflows.
- Help refine onboarding so more contributors can join quickly.
